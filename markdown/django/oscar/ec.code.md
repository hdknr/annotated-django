## JAN(Japanese Article Number)

- 1) 標準7(13桁: 7桁企業 + 5桁商品 + 1チェック) [450 ～ 455 あるいは490 ～ 499で始まる企業コード]
- 2) 標準9(13桁: 9桁企業 + 3桁商品 + 1チェック) [456 ～ 459 で始まる企業コード]
- 3) 短縮(8桁: ６桁企業 + 1桁商品 + 1チェック)
